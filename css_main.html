CSS



Definiţie

Css-ul este un limbaj de stilizare al elementelor html, al tag-urilor html. Denumirea CSS provine din expresia Cascading Style Sheets. În Web Design-ul modern, pentru stilizarea paginilor web se folosește numai CSS. Acest lucru înseamna că de la culoarea literelor și a background-ului până și la pozitionarea elementelor de pe o pagina web, totul este stilizat prin CSS. Un fișier CSS se recunoaște după extensia sa ".css".

Sintaxă:

obiect {

atribute;

atribute; etc.}


Un comentariu în CSS: /*Comentariu */

Există 3 moduri în care se folosește CSS-ul:

1. În tag-ul <head>

Stilul unui document poate fi definit în tag-ul <head> al documentului. Exemplu:

<!DOCTYPE html>
<html>
<head>
<title>Site-ul meu</title>
<style>
body{
background-color: blue;
color: green;
};
</style>
 </head>
<body>
<h1>Bună!</h1>
</body>
</html>

După ce a fost adăugat stilul, pagina va avea fundal albastru și scris verde.

Încearcă!

2. Într-un fișier extern

Un document poate fi stilizat și prin scrierea comenzilor într-un fișier separat CSS, care este mai târziu "legat" de cel HTML în tag-ul <head>. Exemplu:


styel.css
body{
background-color: red;
color: violet;
};
index.html
<!DOCTYPE html>
<html>
<head>
<title>Site-ul meu</title>
<link rel="stylesheet" href="style.css">
 </head>
<body>
<h1>Bună!</h1>
</body>
</html>

Astfel, fundalul site-ului va fi roșu și culoarea scrisului, violet. Această variantă este cea mai recomandată la proiectele mari pentru ca liniile de cod să rămână lizibile și mai usor de parcurs.

Încearcă!

3. În tag-ul elementului

O ultima metoda ar fi stilizarea unui singur element în tag-ul respectiv. Exemplu:

<!DOCTYPE html>
<html>
<head>
<title>Site-ul meu</title>
</head>
<body style="background-color: black; color: white";>
<h1>Bună!</h1>
</body>
</html>

Pagina va fi neagra cu scris alb. Această metodă de stilizare ajută când e nevoie ca un singur element să aibă anumite atribute.

Încearcă!
