Clase si id-uri



Clase

Conform DexOnline, o clasa este un "1. Grup (mare) de obiecte, de elemente, de ființe, de fenomene cu însușiri comune care satisfac un criteriu. 2. (De obicei cu determinarea „socială”) Ansamblu de persoane grupate după criterii economice, istorice și sociologice." De asemenea, aceste elemente pot performa aceleai actiuni/actiuni similare.


Aceste clase ne inconjoara si apar pretutindeni in viata noastra. Unul dintre cele mai cunoscute exemple ar fi clasa de elevi, un "Colectiv de elevi, de obicei de aceeași vârstă și cu aceeași pregătire, care învață împreună în cursul unui an școlar după aceeași programă de învățământ". Toti elevii dintr-o clasa au varste in acelasi interval, acelasi nivel de educatie (de obicei) etc. Pana si scoala, ca un intreg, este o clasa

MARE

care cuprinde obiectele/subclasele "clase de elevi". Toate clasele de elevi apartin aceleiasi institutii, pregatesc copiii in acelasi interval de timp (de obicei), sunt asemanatoare ca structura etc



Clasele din programare merg pe acelasi principiu.

Definirea unei clase constă din două părţi distincte:

  • declararea clasei
  • implementarea clasei respective

Sintaxa in HTML:

<tag class="NumeClasa">

Sintaxa in CSS

.NumeClasa{ atribute; atribute etc.}

ID-uri

ID-ul, prescurtare de la "identification card" (card de identitate) in engleza, reprezinta "un document (cum ar fi un card) care conține informații de identificare, adresea și o fotografie a persoanei al cărei nume apare pe el" conform dictionarului Merriam-Webster.


Un element HTML poate avea un identificator pentru a-l distinge de alte elemente. Acest lucru se face dând un nume unic atributului id al elementului HTML. Nu există două elemente dintr-o pagină web care pot avea aceeași valoare de id.

Orice element, cum ar fi un paragraf, un link sau o imagine, poate avea propriul id unic. Dacă dorim să avem stiluri personalizate bazate pe id-ul elementului, putem folosi un selector CSS special care începe cu simbolul #. Id-ul este util pentru stilizat, dar este util și pentru link.

ID


Sintaxa in HTML:

<tag id="NumeId">

Sintaxa in CSS:

#NumeId{ atribute; atribute etc.}
